System Architecture

🎭
Joke Service Reads jokes from MySQL/MongoDB. Serves this homepage. Stateless — horizontally scalable.
📤
Submit Service Publishes jokes to the RabbitMQ "submit" queue. Never writes to DB directly.
⚖️
Moderate Service Auth0 OIDC-protected. Human approves/rejects jokes from the queue before DB entry.
⚙️
ETL Service Consumes "moderated" queue → writes to DB → emits ECST type_update events.
🐇
RabbitMQ Durable queues, persistent messages, fanout exchange for ECST. At-least-once delivery.
🛡️
Kong Gateway DB-less declarative config. Rate limiting, correlation-id, TLS termination, path routing.
🗄️
MySQL + MongoDB Switchable via DB_TYPE env. Unified adapter pattern across joke-service and ETL.
☁️
Azure + Terraform 4 VMs, VNet, NSGs. Terraform provisions Kong VM. GitHub Actions CI/CD pipeline.
